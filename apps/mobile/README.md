# MGL Store — Mobile App

This directory will house the **Flutter** mobile application for MGL Store (iOS, Android, and Driver App).

## Getting Started

Initialize the Flutter project in this directory:

```bash
flutter create . --project-name mgl_store_mobile --org com.mglstore
```
